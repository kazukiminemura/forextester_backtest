# JPYクロス RR 1:1 共通パラメータ再検証

## 結論

厳格なRR 1:1（SLまでの距離とTPまでの距離を同一、1Rで100%決済）で
JPYクロス共通パラメータを再探索しましたが、実運用候補は採用しません。

学習・検証期間だけで選んだ最上位候補は、未使用の2023～2024年で18取引、
勝率38.89%、純損益-16,200円、PF 0.56でした。探索したリスク条件81候補のうち、
同期間の4通貨合算が純損益プラスかつPF 1超だった候補は0件です。

## 対象と分割

- 主要対象: USDJPY、GBPJPY、AUDJPY、CHFJPY
- 学習: 2004～2018年
- 検証: 2019～2022年
- ホールドアウト: 2023～2024年
- 完全未使用確認: 2025～2026-07-17
- 取引数量: 各通貨0.1 lot
- 約定: Forex Testerの固定スプレッドを反映した1時間足OHLCモデル

EURJPYは履歴が2024-08-27で終了しているため主要4通貨の共通選定から外し、
参考評価だけ行いました。NZDJPYは利用可能な履歴が2018年4月付近に限られるため
対象外です。

候補順位は学習と検証だけで決定し、2023年以降の結果は候補選定に使っていません。

## 通貨別の判断

現在設定の2023～2024年ではGBPJPYだけがプラスで、USDJPY、AUDJPY、CHFJPYは
マイナスです。最上位候補でも同期間にプラスなのはUSDJPYだけでした。
したがって、今回の結果からJPYクロス全体を一つのパラメータで扱うのは適切では
ありません。特にCHFJPYとAUDJPYを同じプリセットで実運用する根拠はありません。

次に進める場合は、共通値をさらにホールドアウトへ合わせるのではなく、通貨ごとに
独立したウォークフォワード検証を行い、最低取引数を満たした通貨だけを有効にする
必要があります。
